Урок 5: Циклы (for, while)

Изучаем, как повторять действия в коде с помощью циклов.

🎯 Цель урока

Научиться использовать циклы for и while для повторения действий.

🔄 Цикл for

Используется для перебора элементов (например, чисел в диапазоне или символов в строке).

for i in range(5):
    print("Шаг №", i) # Выведет от 0 до 4

♾️ Цикл while

Работает до тех пор, пока условие истинно.

count = 0
while count < 3:
    print("Считаю...")
    count += 1

🛑 break и continue

  • break — полностью прерывает цикл.
  • continue — переходит к следующей итерации, пропуская остаток кода в текущей.

📝 Домашнее задание

  • Напиши цикл for, который выведет числа от 1 до 10.
  • Напиши цикл while, который запрашивает у пользователя число, пока он не введёт 0.
  • Используй break для остановки цикла при вводе отрицательного числа.
