Циклы: Автоматизация повторений

Глубокое погружение в тему. Время изучения: ~30 минут.

🎯 Цель урока

Изучить циклы for и while для эффективной обработки данных.

Цикл for

Используется для перебора последовательностей (списков, строк, диапазонов).

for i in range(5):
    print(f"Итерация {i}")

Цикл while

Выполняется, пока условие истинно. Опасайтесь бесконечных циклов!

count = 0
while count < 3:
    print("Работаю...")
    count += 1

Управление циклом

  • break — немедленный выход.
  • continue — переход к следующей итерации.
