Урок 5: Циклы (for, while)

Изучаем, как повторять действия в коде с помощью циклов.

🎯 Цель урока

Научиться использовать циклы for и while для повторения действий, а также управлять ходом цикла с помощью break и continue.

🔄 Цикл for - когда количество повторений известно

Цикл for используется тогда, когда заранее известно количество повторений или есть последовательность элементов для перебора.

for i in range(5):
    print("Привет, мир!")

Этот код 5 раз напечатает «Привет, мир!».

📐 Функция range()

Функция range() генерирует последовательность чисел. Важно знать:

  • range(5) генерирует числа от 0 до 4 (т.е. 0, 1, 2, 3, 4). Число 5 не включается!
  • range(1, 6) - от 1 до 5 включительно.
  • range(0, 10, 2) - чётные числа от 0 до 8 (шаг 2).
for i in range(1, 6):
    print(i)  # Выведет: 1, 2, 3, 4, 5

🔁 Цикл while - пока условие истинно

Цикл while выполняется пока его условие истинно (True). Используется когда заранее неизвестно, сколько раз нужно повторить действие.

count = 0
while count < 5:
    print("Повтор №", count)
    count += 1

Этот код будет выполняться, пока count меньше 5. После каждой итерации (count += 1) значение увеличивается. Когда count станет равным 5 - условие станет ложным и цикл остановится.

⚠️ Осторожно: бесконечный цикл! Если условие никогда не станет ложным, цикл будет работать вечно и «заморозит» программу. Всегда следите, чтобы условие когда-нибудь стало False.

⏹️ Оператор break - досрочный выход из цикла

break немедленно прерывает цикл полностью и переходит к коду после него:

for i in range(10):
    if i == 5:
        break  # Остановит цикл при i = 5
    print(i)
# Выведет: 0, 1, 2, 3, 4

⏩ Оператор continue - пропуск итерации

continue пропускает оставшуюся часть текущей итерации и переходит к следующей. В отличие от break - цикл не останавливается, продолжает работу:

for i in range(10):
    if i % 2 == 0:
        continue  # Пропустит чётные числа
    print(i)
# Выведет: 1, 3, 5, 7, 9 (только нечётные)

  • Напиши цикл while, который запрашивает у пользователя число, пока он не введёт 0.
  • Используй break для остановки цикла при вводе отрицательного числа.
  • Напиши программу, которая с помощью continue выводит только чётные числа от 1 до 20.
