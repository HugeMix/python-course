Урок 5: Циклы (for, while)

Изучаем, как повторять действия в коде с помощью циклов.

🎯 Цель урока

Научиться использовать циклы for и while для повторения действий.

🔄 Цикл for

Цикл for используется, когда заранее известно количество повторений.

for i in range(5):
    print("Привет, мир!")

Этот код 5 раз напечатает "Привет, мир!".

🔁 Цикл while

Цикл while выполняется, пока условие истинно.

count = 0
while count < 5:
    print("Повтор №", count)
    count += 1

Этот код будет выполняться, пока count меньше 5.

⏩ Операторы break и continue

break прерывает цикл, а continue пропускает оставшуюся часть итерации.

for i in range(10):
    if i == 5:
        break  # Остановит цикл при i = 5
    print(i)
for i in range(10):
    if i % 2 == 0:
        continue  # Пропустит четные числа
    print(i)
\n

💡 Подробнее о continue

Оператор continue заставляет программу мгновенно прервать текущий круг выполнения цикла. Но в отличие от break он не убивает цикл, а сразу переходит к следующему шагу!
