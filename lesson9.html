Урок 9: Работа со строками

Изучаем, как работать со строками в Python.

🎯 Цель урока

Научиться работать со строками, изменять их и использовать встроенные методы.

📝 Основы строк

Строки в Python обозначаются в кавычках:

text = "Привет, мир!"
print(text)

🔠 Методы строк

Python предоставляет множество методов для работы со строками:

text = "python"
print(text.upper())  # Преобразует в верхний регистр
print(text.lower())  # Преобразует в нижний регистр
print(text.replace("y", "i"))  # Заменяет символы

✂️ Разделение и объединение строк

words = "apple,banana,orange".split(",")
print(words)
print(" - ".join(words))  # Объединение строк
\n

💡 Полезные свойства строк

  • s.strip() очищает переменную от мусора: переносит пробелы и невидимые символы строго по краям текста.
  • s.replace('a','b') используется для автозамены: меняет все 'a' на 'b'.
