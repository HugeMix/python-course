Урок 9: Работа со строками

Изучаем методы строк, форматирование и полезные приёмы работы с текстом.

🎯 Цель урока

Научиться работать со строками: изменять регистр, искать, заменять, разделять и форматировать текст.

📝 Основы строк

Строки в Python - это текст в кавычках. Можно использовать одинарные или двойные кавычки:

text = "Привет, мир!"
print(text)

# Многострочная строка - тройные кавычки:
poem = """Первая строка
Вторая строка
Третья строка"""
print(poem)

🔠 Методы изменения регистра

text = "python"
print(text.upper())       # Верхний регистр: PYTHON
print(text.lower())       # Нижний регистр: python
print(text.capitalize())  # Первая буква заглавная: Python

🔍 Поиск и замена

text = "python"
print(text.replace("y", "i"))  # Заменяет символы: pithon

sentence = "Я люблю Python"
print(sentence.find("Python"))   # Возвращает индекс начала: 8
print("Python" in sentence)      # Проверяет наличие: True

Метод replace('что', 'на что') заменяет все вхождения первой подстроки на вторую.

📏 Длина строки

text = "python"
print(len(text))  # Длина строки: 6

Функция len() - это встроенная функция (не метод!), она возвращает количество символов в строке.

✂️ Разделение и объединение строк

words = "apple,banana,orange".split(",")
print(words)  # ['apple', 'banana', 'orange']

# Объединение списка строк в одну:
print(" - ".join(words))  # apple - banana - orange

Метод split(разделитель) разбивает строку на список подстрок по указанному разделителю. Метод join(список) - обратная операция.

🧹 Очистка строк от лишних пробелов

Метод strip() удаляет пробелы и невидимые символы строго по краям строки (начало и конец):

messy = "   Привет!   "
clean = messy.strip()
print(clean)  # Выведет: Привет!

💬 Форматирование строк (f-строки)

Современный и удобный способ вставить переменные в строку - f-строки (f-strings). Поставьте букву f перед кавычками и вставляйте переменные в фигурные скобки:

name = "Алекс"
age = 25
print(f"Привет! Меня зовут {name} и мне {age} лет.")
# Выведет: Привет! Меня зовут Алекс и мне 25 лет.
