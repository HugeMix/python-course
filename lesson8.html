Урок 8: Словари и множества

Изучаем работу со словарями и множествами в Python.

🎯 Цель урока

Научиться работать со словарями и множествами, использовать их методы для хранения и обработки данных.

📖 Словари (dict) в Python

Словарь - это структура данных, которая хранит пары ключ-значение в фигурных скобках. Значение извлекается по ключу, а не по индексу:

student = {"имя": "Алекс", "возраст": 20, "город": "Москва"}
print(student["имя"])      # Выведет: Алекс
print(student["возраст"])  # Выведет: 20

🔑 Методы словарей

Основные методы для работы со словарями:

  • keys() - возвращает все ключи словаря.
  • values() - возвращает все значения словаря.
  • items() - возвращает все пары ключ-значение.
  • get(ключ) - безопасно получает значение. Если ключа нет - возвращает None, а не ошибку. 
student = {"имя": "Алекс", "возраст": 20}

print(student.keys())    # dict_keys(['имя', 'возраст'])
print(student.values())  # dict_values(['Алекс', 20])

# Безопасное получение:
print(student.get("город"))  # Выведет None (ключа нет, но без ошибки!)
print(student.get("город", "Не указан"))  # Выведет: Не указан

➕ Добавление и удаление элементов словаря

student = {"имя": "Алекс"}

# Добавление / изменение значения по ключу:
student["возраст"] = 21

# Удаление ключа и его значения:
del student["возраст"]   # удаляет ключ 'возраст' полностью

print(student)  # {'имя': 'Алекс'}

🔢 Множества (set)

Множество - это структура данных, которая хранит только уникальные элементы без повторений. Порядок элементов не гарантирован:

numbers = {1, 2, 3, 4, 5, 5}
print(numbers)  # Выведет: {1, 2, 3, 4, 5}  - дублирующийся 5 удалён!

🛠️ Операции с множествами

  • add() - добавить элемент.
  • remove() - удалить элемент (ошибка если нет).
  • union() - объединение двух множеств.
  • intersection() - пересечение (общие элементы).

⚠️ Важно: как создать пустое множество

Нельзя создать пустое множество через {} - это создаст пустой словарь! Для пустого множества нужно использовать set():

empty_dict = {}       # Это пустой СЛОВАРЬ
empty_set = set()     # Это пустое МНОЖЕСТВО
