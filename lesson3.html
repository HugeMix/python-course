Урок 3: Операторы в Python

Изучаем арифметические, логические и операторы сравнения.

🎯 Цель урока

Познакомиться с основными операторами в Python и научиться их использовать.

➕ Арифметические операторы

Арифметические операторы позволяют выполнять математические операции:

print(10 + 5)  # Сложение
print(10 - 5)  # Вычитание
print(10 * 5)  # Умножение
print(10 / 5)  # Деление
print(10 // 3) # Целочисленное деление
print(10 % 3)  # Остаток от деления
print(2 ** 3)  # Возведение в степень

🔄 Операторы сравнения

Используются для сравнения значений:

print(10 > 5)   # Больше
print(10 < 5)   # Меньше
print(10 == 5)  # Равно
print(10 != 5)  # Не равно
print(10 >= 5)  # Больше или равно
print(10 <= 5)  # Меньше или равно

🧠 Логические операторы

Позволяют комбинировать условия:

a = True
b = False
print(a and b)  # Логическое И
print(a or b)   # Логическое ИЛИ
print(not a)    # Логическое НЕ
\n

💡 Важно узнать!

Помимо представленных выше вычислений, существуют Логические операторы (and, or, not). Они возвращают True или False: not просто меняет логическое состояние любого значения на противоположное (True на False и наоборот).
