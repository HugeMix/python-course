Урок 3: Операторы в Python

Изучаем арифметические, логические и операторы сравнения.

🎯 Цель урока

Познакомиться с основными операторами в Python и научиться их использовать.

➕ Арифметические операторы

Python умеет выполнять все базовые математические операции:

  • + Сложение
  • - Вычитание
  • * Умножение
  • / Деление (всегда дает float)
  • // Целочисленное деление (отбрасывает остаток)
  • % Остаток от деления
  • ** Возведение в степень

⚖️ Операторы сравнения

Результатом сравнения всегда является True или False.

print(5 > 3)  # True
print(10 == 10) # True (равенство)
print(7 != 8)  # True (неравенство)

  • Используйте логические операторы, чтобы проверить, попадает ли число в диапазон от 10 до 50.
