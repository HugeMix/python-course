Урок 3: Операторы в Python

Изучаем арифметические, логические операторы и операторы сравнения.

🎯 Цель урока

Познакомиться с основными операторами в Python и научиться применять их в программах.

➕ Арифметические операторы

Арифметические операторы позволяют выполнять математические действия над числами:

print(10 + 5)   # Сложение → 15
print(10 - 5)   # Вычитание → 5
print(10 * 5)   # Умножение → 50
print(10 / 5)   # Деление → 2.0 (результат всегда float!)
print(10 // 3)  # Целочисленное деление → 3 (отбрасывает дробную часть)
print(10 % 3)   # Остаток от деления → 1
print(2 ** 3)   # Возведение в степень → 8

Обратите внимание: оператор // (целочисленное деление) возвращает только целую часть. Например, 10 // 3 = 3, а не 3.333. Оператор ** - это возведение в степень (в Python не используют ^ для этого).

🔄 Операторы сравнения

Используются для сравнения значений. Всегда возвращают True или False:

print(10 > 5)    # Больше → True
print(10 < 5)    # Меньше → False
print(10 == 5)   # Равно → False
print(10 != 5)   # Не равно → True
print(10 >= 10)  # Больше или равно → True
print(10 <= 5)   # Меньше или равно → False

Важно: для проверки равенства используется двойной знак равно ==, а одинарный = - это присваивание значения переменной.

🧠 Логические операторы

Логические операторы позволяют комбинировать несколько условий:

a = True
b = False
print(a and b)   # Логическое И: True только если ОБА True → False
print(a or b)    # Логическое ИЛИ: True если хотя бы ОДНО True → True
print(not a)     # Логическое НЕ: меняет на противоположное → False

Логические операторы возвращают True или False. Правила:

  • and - правда только если оба условия истинны.
  • or - правда если хотя бы одно условие истинно.
  • not - меняет логическое состояние на противоположное: из True делает False и наоборот.

💡 Операторы присваивания

Помимо обычного =, есть сокращённые операторы присваивания:

x = 10
x += 5   # То же самое что x = x + 5 → x станет 15
x -= 3   # x = x - 3 → x станет 12
x *= 2   # x = x * 2 → x станет 24
x //= 4  # x = x // 4 → x станет 6
