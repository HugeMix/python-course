Урок 4: Условные конструкции: Логика выбора

⏱ Время изучения: ~30 минут 📊 Уровень: Базовый

1. Оператор if

Условный оператор позволяет выполнять код только при соблюдении условия. Важно: В Python отступы определяют вложенность кода.

if age >= 18:
    print("Доступ разрешен")

2. Конструкция else и elif

Если условие if ложно, сработает else. Если нужно проверить несколько условий — используйте elif (сокращение от else if).

3. Вложенные условия

Вы можете помещать один if внутрь другого, но старайтесь не делать слишком большую вложенность, чтобы код оставался читаемым.

4. Тернарный оператор

Краткая запись условия в одну строку: result = "Да" if x > 0 else "Нет".

5. Операторы is и in

  • is: Проверяет идентичность (ссылаются ли переменные на один объект).
  • in: Проверяет наличие элемента в последовательности (например, букву в строке).

6. Значения, которые считаются False

В Python не только False является ложью. Ложными считаются: None, 0, 0.0, пустые строки "", пустые списки [] и словари {}.
