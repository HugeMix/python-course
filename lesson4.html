Урок 4: Условные операторы (if, else, elif)

Как программа может принимать решения и выполнять разные действия в зависимости от условий.

🎯 Цель урока

Познакомиться с конструкцией if-else, научиться использовать условия в программах.

📃 Что такое условные операторы?

Иногда программа должна выполнять разные действия в зависимости от ситуации. Для этого используются условные операторы. Они позволяют задать вопрос: «Если это условие выполняется - делай одно, иначе - другое».

✅ Оператор if - простая проверка условия

Синтаксис условия if:

age = 18
if age >= 18:
    print("Вы совершеннолетний!")

Если условие age >= 18 верно (True), выполняется код внутри блока if. Важно: после условия всегда ставится двоеточие (:), а код внутри блока пишется с отступом в 4 пробела.

➖ Добавляем else - ветка «иначе»

Блок else выполняется тогда, когда условие if оказалось ложным:

age = 16
if age >= 18:
    print("Вы совершеннолетний!")
else:
    print("Вы ещё несовершеннолетний.")

В данном примере, так как age = 16, выполнится ветка else.

🔄 Проверка нескольких условий с elif

elif - сокращение от «else if» («иначе если»). Используется для проверки дополнительных условий, когда первое оказалось ложным. Можно использовать несколько elif подряд:

age = 10
if age >= 18:
    print("Вы взрослый.")
elif age >= 13:
    print("Вы подросток.")
else:
    print("Вы ребёнок.")

Python проверяет условия по очереди сверху вниз и выполняет первую подошедшую ветку.

📈 Логические операторы в условиях

Можно проверять несколько условий одновременно, используя логические операторы из прошлого урока:

temp = 25
if temp > 20 and temp < 30:
    print("Погода хорошая!")
  • and - оба условия должны быть верны.
  • or - хотя бы одно условие должно быть верно.
  • not - меняет значение на противоположное.

💡 Синтаксические правила

Запомните два обязательных правила Python для условий:

  • После if, elif или else всегда ставится двоеточие (:).
  • Код внутри блока условия обязательно пишется с одинаковым отступом - 4 пробела. Это не стиль, а требование языка: их отсутствие вызовет ошибку.

🧪 Проверка истинности значений

В Python условие может быть не только сравнением, но и просто значением. Следующие значения считаются «ложными» (False): 0, "" (пустая строка), [] (пустой список), None. Всё остальное - «истинное»: 

name = ""
if name:
    print("Имя введено:", name)
else:
    print("Имя не указано!")
