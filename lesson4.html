Урок 4: Условные операторы (if, else, elif)

Как программа может принимать решения и выполнять разные действия в зависимости от условий.

🎯 Цель урока

Познакомиться с конструкцией if-else, научиться использовать условия в программах.

🤔 Условный оператор if

Позволяет программе принимать решения в зависимости от условий.

age = 18
if age >= 18:
    print("Доступ разрешен")
else:
    print("Доступ запрещен")

🧩 Конструкция if-elif-else

Если условий много, используйте elif (сокращение от else if).

score = 85
if score >= 90:
    print("Отлично")
elif score >= 70:
    print("Хорошо")
else:
    print("Нужно подтянуть")
⚠️ Важно: Не забывайте про двоеточие : в конце условия и отступы (4 пробела) для блока кода внутри!
