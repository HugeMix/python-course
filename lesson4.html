Урок 4: Условные операторы (if, else, elif)

Как программа может принимать решения и выполнять разные действия в зависимости от условий.

🎯 Цель урока

Познакомиться с конструкцией if-else, научиться использовать условия в программах.

📃 Что такое условные операторы?

Иногда программа должна выполнять разные действия в зависимости от ситуации. Для этого используются **условные операторы**.

✅ Простая проверка условия

Синтаксис **if**:

age = 18
if age >= 18:
    print("Вы совершеннолетний!")

Если условие age >= 18 верно, выполняется код внутри if. Важно использовать отступ (4 пробела).

➖ Добавляем else

age = 16
if age >= 18:
    print("Вы совершеннолетний!")
else:
    print("Вы ещё несовершеннолетний.")

Если условие **ложное**, выполняется код внутри else.

🔄 Проверка нескольких условий с elif

age = 10
if age >= 18:
    print("Вы взрослый.")
elif age >= 13:
    print("Вы подросток.")
else:
    print("Вы ребёнок.")

elif позволяет проверить **дополнительные условия**.

📈 Логические операторы в условиях

Можно проверять **несколько условий** одновременно:

temp = 25
if temp > 20 and temp < 30:
    print("Погода хорошая!")
  • and – оба условия должны быть верны.
  • or – хотя бы одно условие должно быть верно.
  • not – меняет значение на противоположное.
\n

💡 Синтаксическая особенность

После условий if, elif или else всегда должно ставиться двоеточие (:). После него код обязательно пишется с отступом в 4 пробела.
