Урок 12: Модули и библиотеки

Изучим, как подключать модули и использовать библиотеки в Python.

🎯 Цель урока

Научиться использовать модули и библиотеки в Python, загружать и устанавливать их.

📌 Что такое модули?

Модуль - это файл с кодом Python, который можно подключить в программу. Например, модуль math содержит математические функции.

🔧 Подключение встроенных модулей

В Python есть много встроенных модулей. Например, модуль math для работы с математикой:

import math

print(math.sqrt(25))  # Выведет 5.0
print(math.pi)  # Выведет 3.141592653589793

📦 Установка сторонних библиотек

Python поддерживает сторонние библиотеки, которые можно установить с помощью pip. Например, библиотека requests для работы с интернет-запросами: 

pip install requests

Пример использования:

import requests

response = requests.get("https://api.github.com")
print(response.status_code)  # Выведет 200, если запрос успешен
\n

💡 Официальный менеджер Pip

PIP — это официальный менеджер пакетов в Python (Package Installer). При помощи простой команды в терминале: pip install libraries_name вы загружаете сторонний код из интернета на ваш ПК и можете интегрировать его за секунду.
