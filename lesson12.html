Урок 12: Модули и библиотеки

Изучим, как подключать модули и использовать библиотеки в Python.

🎯 Цель урока

Научиться использовать модули и библиотеки в Python, загружать и устанавливать их.

📦 Модули и библиотеки

Не нужно изобретать велосипед. Используйте готовые модули.

import math
print(math.sqrt(16)) # 4.0

import random
print(random.randint(1, 10)) # Случайное число

🐍 Менеджер пакетов PIP

Позволяет устанавливать миллионы сторонних библиотек.

pip install requests

📝 Домашнее задание

  • Подключи модуль random и выведи случайное число.
  • Установи библиотеку requests и попробуй сделать HTTP-запрос.
