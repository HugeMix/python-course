Урок 12: Модули и библиотеки

Изучим, как подключать модули и использовать готовые библиотеки в Python.

🎯 Цель урока

Научиться использовать встроенные модули и сторонние библиотеки, подключать нужные инструменты с помощью import и устанавливать новые пакеты через pip.

📌 Что такое модуль?

Модуль - это файл с готовым кодом Python (.py), который можно подключить в свою программу с помощью ключевого слова import. В Python огромное количество встроенных модулей - они уже установлены и готовы к использованию.

🔧 Подключение встроенных модулей

Пример: модуль math содержит математические функции:

import math

print(math.sqrt(25))  # Квадратный корень → 5.0
print(math.pi)        # Константа π → 3.141592653589793
print(math.ceil(4.1)) # Округление вверх → 5

🎯 Импорт конкретных функций - from ... import

Можно импортировать только нужные функции из модуля. Тогда не нужно каждый раз писать название модуля через точку:

from math import pi, sqrt

print(sqrt(16))  # Выведет 4.0
print(pi)        # Выведет 3.141592653589793

🎲 Модуль random - случайные числа

Встроенный модуль random используется для генерации случайных чисел:

import random

print(random.randint(1, 10))  # Случайное целое число от 1 до 10
print(random.random())        # Случайное число от 0.0 до 1.0

fruits = ["яблоко", "банан", "вишня"]
print(random.choice(fruits))  # Случайный элемент из списка

📦 Установка сторонних библиотек - pip

pip (Package Installer for Python) - это официальный менеджер пакетов Python. С его помощью можно в одну команду в терминале загрузить и установить любую стороннюю библиотеку из интернета:

pip install requests

После установки библиотеку можно использовать как обычный модуль:

import requests

response = requests.get("https://api.github.com")
print(response.status_code)  # Выведет 200, если запрос успешен

💡 Псевдоним при импорте - as

Длинному имени модуля можно дать короткий псевдоним с помощью ключевого слова as:

import math as m

print(m.sqrt(9))  # 3.0 - всё так же, но через псевдоним `m`
