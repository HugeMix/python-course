Урок 12: Модули и пакеты: Расширяем горизонты

⏱ Время изучения: ~30 минут 📊 Уровень: Базовый

1. Импорт модулей

Модуль — это просто файл с кодом .py. Импорт позволяет использовать его функции.

  • import math
  • from math import sqrt
  • import numpy as np (псевдоним)

2. Стандартная библиотека

Python поставляется с "батарейками в комплекте":

  • os, sys — работа с системой.
  • math, random — математика и случайность.
  • datetime — время.
  • json — работа с данными.

3. Менеджер пакетов pip

Позволяет устанавливать библиотеки из внешнего репозитория PyPI: pip install requests.

4. Создание своего модуля

Просто создайте файл utils.py и импортируйте его в главном файле import utils.

5. Конструкция if __name__ == "__main__":

Позволяет коду запускаться только при прямом вызове файла, но не при его импорте как модуля.

6. Виртуальные окружения (venv)

Всегда используйте venv для разных проектов, чтобы зависимости одного проекта не конфликтовали с другим.
