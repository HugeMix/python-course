Урок 15: Итоговый проект - To-Do List

Примените все полученные знания и создайте своё первое полноценное приложение.

🎯 Задача

Создайте консольное приложение на Python, которое поможет пользователю вести список дел (To-Do List) с сохранением данных в файл.

📌 Описание проекта

Программа должна позволять:

  • Добавлять задачи
  • Просматривать список задач
  • Удалять выполненные задачи
  • Сохранять список в файл и загружать его при следующем запуске

💻 Пример кода

import json

def load_tasks():
    try:
        with open("tasks.json", "r", encoding="utf-8") as file:
            return json.load(file)
    except FileNotFoundError:
        return []

def save_tasks(tasks):
    with open("tasks.json", "w", encoding="utf-8") as file:
        json.dump(tasks, file, ensure_ascii=False, indent=4)

def main():
    tasks = load_tasks()
    while True:
        print("\n1. Добавить задачу")
        print("2. Показать список задач")
        print("3. Удалить задачу")
        print("4. Выйти")
        choice = input("Выберите действие: ")

        if choice == "1":
            task = input("Введите новую задачу: ")
            tasks.append(task)
            save_tasks(tasks)
        elif choice == "2":
            print("\nВаш список дел:")
            for i, task in enumerate(tasks, 1):
                print(f"{i}. {task}")
        elif choice == "3":
            num = int(input("Введите номер задачи для удаления: ")) - 1
            if 0 <= num < len(tasks):
                tasks.pop(num)
                save_tasks(tasks)
            else:
                print("Некорректный номер!")
        elif choice == "4":
            break
        else:
            print("Неверный ввод, попробуйте снова.")

if __name__ == "__main__":
    main()

🔍 Какие знания курса задействованы?

  • Функции (def) - load_tasks(), save_tasks(), main()
  • Списки - tasks.append(), tasks.pop(), перебор с enumerate()
  • Файлы - with open(), чтение и запись JSON
  • Обработка ошибок - try-except FileNotFoundError
  • Циклы - бесконечный while True с выходом через break
  • Условия - if-elif-else
  • Строки - f-строки для форматирования вывода

🔁 Паттерн while True + break - бесконечное меню

В итоговом проекте главное меню реализовано через конструкцию while True. Цикл выполняется бесконечно, пока пользователь не выберет «Выйти» - тогда срабатывает break и цикл останавливается:

while True:
    choice = input("Введите 1, 2 или 3: ")
    if choice == "3":
        break  # Выход из бесконечного цикла
    print("Продолжаем...")

Это стандартный паттерн для консольных меню в Python.

🔢 Функция enumerate() - нумерованный перебор

Функция enumerate(список) позволяет одновременно получать индекс и значение каждого элемента при переборе - очень удобна для вывода нумерованных списков:

tasks = ["Купить молоко", "Позвонить маме", "Сделать зарядку"]
for i, task in enumerate(tasks, 1):  # start=1 начинает нумерацию с 1
    print(f"{i}. {task}")
# Выведет:
# 1. Купить молоко
# 2. Позвонить маме
# 3. Сделать зарядку

Именно так в проекте выводится пронумерованный список задач: for i, task in enumerate(tasks, 1).
