Урок 10: Файлы в Python

Научимся работать с файлами: читать, записывать и безопасно закрывать их.

🎯 Цель урока

Понять, как открывать, читать и записывать файлы в Python, и научиться делать это безопасно.

📂 Открытие файла - функция open()

Встроенная функция open() открывает файл и возвращает объект для работы с ним. Принимает имя файла и режим работы:

file = open("example.txt", "r")  # "r" = режим чтения
content = file.read()            # Читаем весь файл в строку
print(content)
file.close()                     # ОБЯЗАТЕЛЬНО закрываем файл!

📋 Режимы открытия файлов

  • "r" - чтение (файл должен существовать)
  • "w" - запись (перезапишет файл с нуля)
  • "a" - дозапись в конец файла (данные сохраняются)
  • "x" - создание нового файла (ошибка, если уже существует)

✏️ Запись в файл

Открываем файл в режиме записи "w" и записываем текст:

file = open("example.txt", "w")
file.write("Привет, мир!\n")
file.write("Это вторая строка.\n")
file.close()

📌 Чтение построчно

Читаем файл построчно - экономно для больших файлов:

file = open("example.txt", "r")
for line in file:
    print(line.strip())  # strip() убирает лишний перевод строки
file.close()

Метод readlines() читает все строки и возвращает их в виде списка:

file = open("example.txt", "r")
lines = file.readlines()
file.close()
print(lines)  # ['Привет, мир!\n', 'Это вторая строка.\n']

🔒 Безопасный способ: with open

Конструкция with open() - рекомендуемый способ работы с файлами. Файл закрывается автоматически по выходу из блока, даже если произошла ошибка:

with open("example.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
# Файл автоматически закрывается здесь!

Писать file.close() вручную ненадёжно - при ошибке файл может остаться «открытым» в памяти. Блок with open(...) as f: решает эту проблему полностью.

⚙️ Кодировка - параметр encoding

При работе с файлами, содержащими русский текст, всегда указывайте кодировку encoding="utf-8", чтобы символы читались и записывались корректно: 

with open("data.txt", "w", encoding="utf-8") as f:
    f.write("Привет, это русский текст!")
