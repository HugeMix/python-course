Урок 10: Файлы в Python

Научимся работать с файлами: читать и записывать данные.

🎯 Цель урока

Понять, как открывать, читать и записывать файлы в Python.

📂 Открытие файла

Используем функцию open() для работы с файлами.

file = open("example.txt", "r")
content = file.read()
print(content)
file.close()

✏️ Запись в файл

Открываем файл в режиме записи "w" и записываем текст.

file = open("example.txt", "w")
file.write("Привет, мир!")
file.close()

📌 Чтение построчно

Читаем файл построчно с помощью readline() или readlines().

file = open("example.txt", "r")
for line in file:
    print(line.strip())
file.close()
\n

💡 Секреты файлов: with open

Писать file.close() вручную крайне ненадёжно, так как при любой ошибке файл останется висеть в памяти. Безопаснее применять блок with open(...) as f:, который гарантирует, что файл автоматически закроется по окончании работы вашего кода.
