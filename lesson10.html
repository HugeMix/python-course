Урок 10: Работа с файлами: Постоянное хранение

⏱ Время изучения: ~30 минут 📊 Уровень: Базовый

1. Открытие файлов (open)

Функция open(filename, mode). Режимы:

  • 'r': Чтение (по умолчанию).
  • 'w': Запись (стирает старое!).
  • 'a': Дозапись (в конец).
  • 'b': Бинарный режим (для картинок).

2. Конструктор with (Context Manager)

Всегда используйте with. Он гарантирует закрытие файла, даже если в коде произошла ошибка. Забытый открытый файл может привести к утечке памяти.

3. Чтение данных

  • read(): Весь файл одной строкой.
  • readline(): По одной строке.
  • readlines(): Список всех строк.

4. Запись данных

Методы write(string) и writelines(list). Помните, что write не добавляет перенос строки сам!

5. Кодировки

Всегда указывайте encoding='utf-8', чтобы избежать проблем с русскими буквами на разных ОС.

6. Работа с путями (os.path и pathlib)

Для профессиональной работы с файловой системой используйте модуль pathlib. Он позволяет работать с путями как с объектами.
