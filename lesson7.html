Урок 7: Функции в Python

Изучаем, как создавать и использовать функции для удобства программирования.

🎯 Цель урока

Научиться создавать функции, передавать в них аргументы, получать результаты и использовать значения по умолчанию.

📈 Что такое функция?

Функция - это именованный блок кода, который выполняет определённую задачу и может быть вызван в программе несколько раз. Функции помогают избежать повторения кода (принцип DRY - Don't Repeat Yourself).

✅ Создание функции с помощью def

Для создания функции используется ключевое слово def:

def greet():
    print("Привет! Добро пожаловать в Python!")

greet()  # Вызов функции - выведет приветствие

После слова def указывается имя функции, затем круглые скобки () и обязательное двоеточие :. Тело функции пишется с отступом.

🤖 Аргументы (параметры) функции

Функция может принимать данные через аргументы (они же параметры) и использовать их внутри:

def greet(name):
    print("Привет,", name, "!")

greet("Алекс")   # Выведет: Привет, Алекс !
greet("Мария")   # Выведет: Привет, Мария !

Функция с аргументами становится универсальной - одна функция работает для разных данных.

🌟 Возвращаемое значение - оператор return

Функция может возвращать результат своей работы с помощью оператора return. Этот результат можно сохранить в переменную: 

def square(number):
    return number * number

result = square(5)
print("Квадрат числа 5 равен", result)  # Выведет: 25

Важно: функция без return автоматически возвращает None - специальный объект «ничего». Это нормально, если функция просто что-то делает, а не считает.

💡 Значения по умолчанию

Параметр функции может иметь значение по умолчанию. Если при вызове аргумент не передан, Python подставит это значение автоматически:

def greet(name='Гость'):
    print("Привет,", name, "!")

greet()           # Выведет: Привет, Гость !
greet("Дмитрий") # Выведет: Привет, Дмитрий !

📦 Несколько аргументов

Функция может принимать несколько аргументов через запятую:

def add(a, b):
    return a + b

print(add(3, 7))   # Выведет: 10
print(add(10, 20)) # Выведет: 30
