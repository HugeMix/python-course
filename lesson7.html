Урок 7: Функции в Python

Изучаем, как создавать и использовать функции для удобства программирования.

🎯 Цель урока

Научиться создавать функции, передавать в них аргументы и получать результаты.

📈 Что такое функция?

Функция - это блок кода, который выполняет определённую задачу и может вызываться в программе несколько раз.

✅ Создание функции

Используем ключевое слово def для создания функции.

def приветствие():
    print("Привет! Добро пожаловать в Python!")

приветствие()  # Вызов функции

🤖 Аргументы функции

Функция может принимать данные (аргументы) и использовать их внутри.

def приветствие(имя):
    print("Привет,", имя, "!")

приветствие("Алекс")

🌟 Возвращаемое значение

Функция может возвращать результат с помощью return.

def квадрат(число):
    return число * число

результат = квадрат(5)
print("Квадрат числа 5 равен", результат)
\n

💡 Значения по-умолчанию

Функции могут иметь запасные стандарты переменных: например, def greet(name='Гость'):. Если вызовем greet() пустым (без аргументов), то Python по-умолчанию подставит имя: 'Гость'. Вы также должны знать, что функция без return автоматически вернет пустоту — объект None.
