Урок 7: Функции в Python

Изучаем, как создавать и использовать функции для удобства программирования.

🎯 Цель урока

Научиться создавать функции, передавать в них аргументы и получать результаты.

🛠️ Создание функций

Функции позволяют группировать код для повторного использования.

def say_hello(name):
    print("Привет,", name)

say_hello("Максим")

📥 Аргументы и возвращаемое значение

Используйте return, чтобы функция отдала результат обратно в программу.

def add(a, b):
    return a + b

result = add(5, 7) # result станет 12
