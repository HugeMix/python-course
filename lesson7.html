Урок 7: Функции: Повторное использование кода

⏱ Время изучения: ~30 минут 📊 Уровень: Базовый

1. Определение функции

Функции помогают избежать дублирования кода. Используйте ключевое слово def.

def greet(name):
    return f"Привет, {name}!"

2. Аргументы и параметры

  • Позиционные: Порядок важен.
  • Именованные: func(name="Ivan").
  • По умолчанию: def func(a=10).

3. Возврат значения (return)

Если в функции нет return, она автоматически возвращает None. Функция прекращает выполнение сразу после вызова return.

4. Область видимости (Scope)

  • Локальные переменные: Внутри функции.
  • Глобальные: Вне функций. Используйте global только в крайних случаях!

5. Лямбда-функции (Анонимные)

Короткие функции в одну строку: add = lambda x, y: x + y. Удобны для сортировок и фильтраций.

6. *args и **kwargs

Позволяют функции принимать любое количество аргументов. *args — список, **kwargs — словарь.
