Урок 1: Введение в Python

Познакомимся с языком Python, установим среду и напишем первую программу.

🎯 Цель урока

Познакомиться с языком Python, установить среду для программирования и написать первую программу.

📌 Введение

Python — это простой и понятный язык программирования, который используется для создания сайтов, игр, ботов, мобильных приложений и многого другого.

🔧 Установка Python

Чтобы начать программировать, нужно установить Python:

  1. Перейди на официальный сайт: python.org.
  2. Скачай последнюю версию для своей операционной системы.
  3. Установи, поставив галочку "Add Python to PATH" (чтобы Python работал в командной строке).

💻 Проверка установки

Открой **командную строку** (Windows) или **терминал** (Mac/Linux) и напиши:

python --version

Если появится версия Python, значит всё установлено правильно.

🚀 Первая программа

Открой **Python IDLE** (или любой текстовый редактор) и напиши:

print("Привет, мир!")

Запусти код — если на экране появилось Привет, мир!, значит всё работает! 🎉

Важно о функции print():

  • Она выполняет одну роль: выводит переданные данные в консоль (на экран).
  • Python регистрозависим! Это значит, что функции print и PRiNT — это разные вещи. Команды нужно писать строго маленькими буквами.

Дополнительная теория:

Кто создал Python? Язык Python был создан программистом по имени Гвидо ван Россум.

В каком году? Первый релиз языка состоялся в 1991 году.

Какое расширение файлов? Все файлы с кодом Python имеют расширение .py.
