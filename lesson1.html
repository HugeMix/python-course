Урок 1: Введение в Python

Познакомимся с языком Python, установим среду и напишем первую программу.

🎯 Цель урока

Познакомиться с языком Python, установить среду для программирования и написать первую программу.

📌 Что такое Python?

Python - это простой и понятный язык программирования, который используется для создания сайтов, игр, ботов, анализа данных, искусственного интеллекта и многого другого. Язык был создан программистом Гвидо ван Россумом, а первый релиз состоялся в 1991 году.

Главные плюсы Python: читаемый синтаксис, похожий на обычный English-текст, огромное сообщество и миллионы готовых библиотек.

🔧 Установка Python

Чтобы начать программировать, нужно установить Python:

  1. Перейди на официальный сайт: python.org.
  2. Скачай последнюю версию для своей операционной системы.
  3. Установи, поставив галочку "Add Python to PATH" - это позволит запускать Python из командной строки.

💻 Проверка установки

Открой командную строку (Windows) или терминал (Mac/Linux) и напиши: 

python --version

Если появится версия Python (например, Python 3.12.0), значит всё установлено правильно.

🚀 Первая программа - Hello World

Открой Python IDLE (устанавливается вместе с Python) или любой текстовый редактор, например VS Code, и напиши:

print("Привет, мир!")

Запусти код - если на экране появилось Привет, мир!, значит всё работает! 🎉

📢 О функции print()

  • Она выполняет одну роль: выводит переданные данные в консоль (на экран).
  • Python регистрозависим! Это значит, что print и PRiNT - это совершенно разные вещи. Команды нужно писать строго маленькими буквами.
  • Текст внутри print() нужно помещать в кавычки - одинарные 'текст' или двойные "текст".

📁 Файлы Python

Все файлы с кодом на Python имеют расширение .py. Например: my_program.py. Чтобы запустить файл, напишите в терминале: 

python my_program.py

💡 Редакторы кода

Для написания программ удобно использовать специальные редакторы:

  • Python IDLE - встроенный редактор, идеален для новичков.
  • VS Code - мощный бесплатный редактор с подсветкой синтаксиса.
  • PyCharm - профессиональная среда разработки.

📝 Домашнее задание

  • Установи Python и проверь его версию командой python --version.
  • Напиши и запусти программу, которая выводит Привет, мир!.
  • Попробуй изменить текст так, чтобы программа выводила твоё имя.
  • Попробуй написать Print("тест") и посмотри, какую ошибку выдаст Python - это наглядно покажет что значит «регистрозависимый».
