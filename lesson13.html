Урок 13: Основы ООП

Познакомимся с объектно-ориентированным программированием в Python.

🎯 Цель урока

Научиться создавать классы и объекты, работать с атрибутами и методами.

📌 Что такое ООП?

Объектно-ориентированное программирование (ООП) - это стиль программирования, где код организован в виде объектов.

🔧 Создание класса

Класс - это шаблон для создания объектов. Давайте создадим класс Car:

class Car:
    def __init__(self, brand, model, year):
        self.brand = brand  # Атрибут марки
        self.model = model  # Атрибут модели
        self.year = year    # Атрибут года выпуска

    def show_info(self):
        print(f"Автомобиль: {self.brand} {self.model}, {self.year} года выпуска")

🚗 Создание объекта

Объект - это экземпляр класса. Создадим объект машины:

my_car = Car("Toyota", "Camry", 2020)
my_car.show_info()

Вывод:

Автомобиль: Toyota Camry, 2020 года выпуска

⚙ Методы класса

Методы - это функции внутри класса. Добавим метод start_engine:

class Car:
    def __init__(self, brand, model, year):
        self.brand = brand
        self.model = model
        self.year = year

    def show_info(self):
        print(f"Автомобиль: {self.brand} {self.model}, {self.year} года выпуска")

    def start_engine(self):
        print(f"{self.brand} {self.model}: Двигатель запущен!")

my_car = Car("BMW", "X5", 2022)
my_car.show_info()
my_car.start_engine()

💡 Принцип Инкапсуляции

Одним из китов ООП является Инкапсуляция — это умение объектов утаивать и скрывать свои внутренние данные от внешних пользователей ради безопасности.
