Урок 13: Основы ООП

Познакомимся с объектно-ориентированным программированием в Python.

🎯 Цель урока

Научиться создавать классы и объекты, работать с атрибутами и методами.

🏗️ Основы ООП

Объектно-ориентированное программирование — это способ организации кода через "объекты" (чертежи и детали).

class Dog:
    def __init__(self, name):
        self.name = name
    
    def bark(self):
        print(f"{self.name} говорит: Гав!")

my_dog = Dog("Шарик")
my_dog.bark()
