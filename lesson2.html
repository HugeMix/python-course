Урок 2: Переменные и типы данных: Фундамент Python

⏱ Время изучения: ~30 минут 📊 Уровень: Базовый

1. Что такое переменная?

Переменная в Python — это не "коробка" с данными, а ярлык (ссылка) на объект в памяти. Когда вы пишете x = 5, вы создаете объект "5" и вешаете на него ярлык "x".

2. Основные типы данных

ТипНазваниеПример
intЦелые числа10, -5, 0
floatЧисла с плавающей точкой3.14, -0.001
strСтроки"Привет", 'Python'
boolЛогический типTrue, False

3. Динамическая типизация

В Python тип переменной определяется в момент присваивания. Вы можете сначала написать x = 5, а потом x = "Hello". Это удобно, но требует осторожности.

4. Работа с числами

Python поддерживает длинную арифметику — целые числа могут быть сколь угодно большими, пока хватает памяти. Вы можете легко вычислить 2 ** 1000.

5. Тонкости строк

Строки в Python неизменяемы (immutable). Это значит, что вы не можете изменить символ внутри строки, вы можете только создать новую строку. Используйте f-строки для удобного форматирования: f"Значение: {x}".

6. Преобразование типов (Casting)

Иногда нужно превратить строку в число: int("10"). Или число в строку: str(5.5). Помните, что int("3.14") вызовет ошибку, нужно сначала в float.
