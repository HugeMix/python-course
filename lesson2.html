Урок 2: Переменные и типы данных

Разберёмся с переменными, типами данных и их преобразованием.

🎯 Цель урока

Научиться работать с переменными, понимать их типы и уметь преобразовывать данные из одного типа в другой.

📌 Что такое переменная?

Переменная - это «контейнер» для хранения данных. В Python переменную можно создать, просто присвоив ей значение с помощью знака =:

x = 5          # Создали переменную x и присвоили ей число 5
name = "Анна"  # Создали переменную name и присвоили ей строку "Анна"

В отличие от многих других языков, в Python не нужно объявлять тип переменной - он определяется автоматически по значению.

⚠️ Правила именования переменных

  • Имя может содержать только буквы, цифры и символ подчёркивания (_).
  • Имя не может начинаться с цифры: 1var - недопустимо, вызовет ошибку.
  • Рекомендуется писать переменные строчными буквами, разделяя слова подчёркиванием (стиль snake_case): my_variable, user_name.
  • Python чувствителен к регистру: Name и name - это разные переменные!

🔢 Основные типы данных

Python автоматически определяет тип переменной. Вот четыре основных типа:

  • int (целые числа): age = 20
  • float (числа с плавающей точкой): height = 1.75
  • str (строки - текст в кавычках): city = "Москва"
  • bool (логические значения True/False): is_student = True

🔍 Функция type()

Узнать тип переменной можно с помощью встроенной функции type():

print(type("10"))   # Выведет <class 'str'>
print(type(5))      # Выведет <class 'int'>
print(type(3.14))   # Выведет <class 'float'>
print(type(True))   # Выведет <class 'bool'>

🔄 Преобразование типов

Иногда нужно преобразовать один тип данных в другой. Для этого используют специальные функции:

  • int() - преобразует в целое число
  • float() - в число с плавающей точкой
  • str() - в строку
  • bool() - в логическое значение
num = "10"       # Это строка
num = int(num)   # Теперь это число (целое)
print(num + 5)   # Выведет 15

💡 Конкатенация строк

Если сложить две строки оператором +, они «склеятся» - это называется конкатенацией. Это НЕ математическое сложение! 

print(str(5) + '5')  # Результатом будет '55', а не 10!

Поэтому при работе с числами важно убедиться, что они действительно числа, а не строки.

📥 Ввод данных от пользователя

Чтобы пользователь мог ввести данные с клавиатуры, используют функцию input(). Она всегда возвращает строку, поэтому при необходимости нужно конвертировать тип:

name = input("Как тебя зовут? ")
print("Привет,", name)

age = int(input("Сколько тебе лет? "))  # Конвертируем строку в число
print("Через 10 лет тебе будет:", age + 10)
