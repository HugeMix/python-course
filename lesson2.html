Урок 2: Переменные и типы данных

Разберёмся с переменными, типами данных и их преобразованием.

🎯 Цель урока

Научиться работать с переменными, понимать их типы и уметь преобразовывать данные.

📌 Что такое переменные?

Переменная - это "контейнер" для хранения данных. В Python переменную можно создать, просто присвоив ей значение.

x = 5  # Создали переменную x и присвоили ей число 5
name = "Анна"  # Создали переменную name и присвоили ей строку "Анна"

⚠️ Правила именования переменных:

  • Имя может содержать только буквы, цифры и символ подчеркивания (_).
  • Имя не может начинаться с цифры (например, 1var - недопустимо, вызовет ошибку).
  • Рекомендуется писать переменные строчными буквами, разделяя слова подчеркиванием (Snake Case): my_var. Теги var1 или _var_ тоже допустимы, но первый вариант предпочтительнее.

🔢 Основные типы данных

Python автоматически определяет тип переменной. Вот основные типы данных:

  • int (целые числа): age = 20
  • float (числа с запятой): height = 1.75
  • str (строки): city = "Москва"
  • bool (логические значения): is_student = True

Узнать тип переменной можно с помощью функции type():

print(type("10")) # Выведет <class 'str'>
print(type(5))    # Выведет <class 'int'>

🔄 Преобразование типов

Иногда нужно преобразовать один тип данных в другой:

num = "10"  # Это строка
num = int(num)  # Преобразуем в число

Функции преобразования:

  • int() – преобразует в целое число
  • float() – в число с плавающей точкой
  • str() – в строку
  • bool() – в логическое значение

💡 Зачем нужны преобразования? Пример (Сложение строк):

Представь, что у нас есть число 5, которое мы превратили в строку. Если сложить строку с другой строкой, они просто "склеятся" (это называется конкатенацией):

print(str(5) + '5') # Результатом будет '55', а не 10!
  • Создай переменные для своего имени, возраста и роста.
  • Выведи их на экран с помощью print().
  • Попробуй преобразовать возраст в строку, а рост в целое число.
