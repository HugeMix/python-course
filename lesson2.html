Урок 2: Переменные и типы данных

Разберёмся с переменными, типами данных и их преобразованием.

🎯 Цель урока

Научиться работать с переменными, понимать их типы и уметь преобразовывать данные.

📦 Что такое переменные?

Представьте переменную как коробку с наклеенным именем, в которой лежит какое-то значение. Мы можем обращаться к содержимому коробки по её имени.

name = "Алексей"
age = 25
is_student = True

🔢 Основные типы данных

Тип Описание Пример
int Целые числа 10, -5, 0
float Числа с плавающей точкой 3.14, -0.5
str Строки (текст) "Hello", 'Python'
bool Логический тип True, False

📝 Правила именования

Имя переменной должно начинаться с буквы или подчеркивания. Оно не может начинаться с цифры и не должно содержать пробелов.

  • Создай переменные для своего имени, возраста и роста.
  • Выведи их на экран с помощью print().
  • Попробуй преобразовать возраст в строку, а рост в целое число.
