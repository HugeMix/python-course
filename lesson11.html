Урок 11: Обработка ошибок (try-except)

Научимся перехватывать ошибки в коде и правильно их обрабатывать.

🎯 Цель урока

Понять, как использовать конструкции try-except для обработки ошибок и предотвращения «краша» программы.

❓ Зачем нужна обработка ошибок?

Любая программа может столкнуться с ситуацией, которую не предусмотрел программист: пользователь ввёл текст вместо числа, файл не найден, нет подключения к сети. Без обработки ошибок программа просто «падает». Блок try-except позволяет перехватить ошибку и сообщить о ней пользователю, а не аварийно завершиться.

⚠️ Базовая конструкция try-except

Код, который может вызвать ошибку, помещается в блок try. В блоке except описывается, что делать при возникновении ошибки: 

try:
    число = int(input("Введите число: "))
    print("Вы ввели:", число)
except ValueError:
    print("Ошибка! Нужно ввести число, а не текст.")

Если пользователь введёт буквы вместо числа, возникнет ошибка ValueError, и выполнится блок except вместо «краша».

🔄 Блок finally - выполняется всегда

Блок finally выполняется всегда - и при ошибке, и при успешном выполнении. Удобен для закрытия ресурсов:

try:
    файл = open("data.txt", "r")
    содержимое = файл.read()
    print(содержимое)
except FileNotFoundError:
    print("Ошибка! Файл не найден.")
finally:
    print("Попытка чтения файла завершена.")

🚨 Типы ошибок (исключений) в Python

Популярные встроенные типы ошибок:

  • ValueError - неверное значение (например, int("abc"))
  • ZeroDivisionError - деление на ноль (10 / 0)
  • FileNotFoundError - файл не найден
  • TypeError - операция с неверным типом данных
  • Exception - базовый класс всех ошибок. Перехватывает любую ошибку. Используйте его с осторожностью. 
# Перехватить любую ошибку и вывести её описание:
try:
    result = 10 / 0
except Exception as e:
    print("Произошла ошибка:", e)

🚀 Ключевое слово raise - принудительный вызов ошибки

Иногда нужно намеренно вызвать ошибку, если данные некорректны. Для этого используется ключевое слово raise:

age = -5
if age < 0:
    raise ValueError("Возраст не может быть отрицательным!")
print("Возраст:", age)

Если выполнить этот код, программа выбросит ошибку ValueError с нашим сообщением.

🔗 Несколько блоков except

Можно обработать разные типы ошибок по-разному:

try:
    x = int(input("Введите делимое: "))
    y = int(input("Введите делитель: "))
    print("Результат:", x / y)
except ValueError:
    print("Введите только числа!")
except ZeroDivisionError:
    print("На ноль делить нельзя!")
