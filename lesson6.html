Урок 6: Списки и кортежи: Коллекции данных

⏱ Время изучения: ~30 минут 📊 Уровень: Базовый

1. Списки (List)

Списки — это изменяемые (mutable) упорядоченные коллекции. Они могут содержать элементы разных типов.

my_list = [1, "apple", 3.14]

2. Индексация и срезы (Slices)

Индексы начинаются с 0. Python поддерживает отрицательные индексы (-1 — последний элемент). Срезы позволяют брать части списка: list[start:stop:step].

3. Основные методы списков

  • append(x): Добавить в конец.
  • extend(list): Расширить другим списком.
  • insert(i, x): Вставить по индексу.
  • remove(x): Удалить первый найденный элемент.
  • pop(i): Удалить и вернуть элемент по индексу.

4. Кортежи (Tuple)

Кортежи — это неизменяемые списки. Создаются в круглых скобках (1, 2, 3). Они работают быстрее списков и защищают данные от изменений.

5. Списковые включения (List Comprehensions)

Элегантный способ создания списков: [x**2 for x in range(10)]. Это быстрее и короче обычного цикла.

6. Распаковка (Unpacking)

Вы можете присвоить элементы списка переменным за один раз: a, b = [10, 20].
