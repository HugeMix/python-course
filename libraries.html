Математика / Наука данных

📐 NumPy

Фундаментальная библиотека для научных вычислений в Python. Предоставляет поддержку для работы с многомерными массивами (матрицами) и содержит множество высокоуровневых математических функций для операций над ними. Она работает намного быстрее стандартных списков Python, так как написана на языке C.

import numpy as np

# Создание массива
arr = np.array([1, 2, 3, 4])
print(arr * 2) # Выведет: [2 4 6 8]
Анализ данных

📊 Pandas

Мощный инструмент для работы с табличными данными (Excel, CSV, SQL). Он предоставляет структуры данных Series (одномерные данные) и DataFrame (таблицы), с помощью которых можно легко фильтровать, сортировать и анализировать миллионы строк данных.

import pandas as pd

# Создание таблицы (DataFrame)
data = {"Имя": ["Анна", "Иван"], "Возраст": [20, 25]}
df = pd.DataFrame(data)
print(df)
Визуализация

📉 Matplotlib

Самая популярная библиотека для построения графиков и диаграмм (линейных, столбчатых, круговых, гистограмм). Идеально работает в связке с NumPy и Pandas для визуализации тенденций и результатов анализа данных.

import matplotlib.pyplot as plt

plt.plot([1, 2, 3], [10, 20, 15])
plt.title("Мой первый график")
plt.show()
Веб-сети

🌐 Requests

Позволяет отправлять HTTP-запросы простым и понятным способом. С её помощью можно скачивать страницы из интернета, работать с API других сайтов (например, получать погоду), сохранять картинки и автоматизировать работу в сети.

import requests

response = requests.get("https://api.github.com")
print("Статус:", response.status_code)
Веб-разработка (Фреймворк)

🚀 Django

Полнофункциональный и мощный фреймворк для создания сложных веб-сайтов. Он включает в себя всё необходимое "из коробки" (систему авторизации, панель администратора, работу с базами данных) и используется в таких проектах, как Instagram или Pinterest.

# Установка через терминал
pip install django

# Создание нового проекта
django-admin startproject mywebsite
Веб-разработка (Микрофреймворк)

⚙️ Flask

В отличие от Django, Flask — это легковесный инструмент. Он идеален для написания простых веб-приложений, API или бота для Telegram. Он даёт программисту полный контроль над архитектурой и не навязывает лишних функций.

from flask import Flask
app = Flask(__name__)

@app.route("/")
def home():
    return "Привет, мир Сервера!"

if __name__ == "__main__":
    app.run()
Парсинг (Сбор данных)

🥣 BeautifulSoup

Библиотека для извлечения данных из HTML и XML файлов. Если вам нужно вытащить цены на товары, новости или контакты с чужого сайта (веб-скрапинг), BeautifulSoup позволяет легко находить нужные теги в коде страницы.

from bs4 import BeautifulSoup

html_doc = "<h1>Заголовок</h1>"
soup = BeautifulSoup(html_doc, 'html.parser')
print(soup.h1.text) # Выведет: Заголовок
Создание Игр

🎮 Pygame

Инструмент для написания простых 2D-игр и мультимедийных приложений. Отлично подходит для новичков, чтобы изучить концепции программирования (события, циклы, координаты) на примере разработки собственной игры вроде Тетриса или Змейки.

import pygame

pygame.init()
screen = pygame.display.set_mode((400, 300))
pygame.display.set_caption("Моя игра")
Нейросети / AI

🧠 TensorFlow

Одна из самых мощных библиотек машинного обучения, разработанная Google. Используется для создания нейронных сетей, распознавания лиц на фотографиях, анализа текста и создания искусственного интеллекта.

import tensorflow as tf

# Пример создания базовой модели:
model = tf.keras.models.Sequential([
    tf.keras.layers.Dense(10, activation='relu')
])
Графика

🖼️ Pillow (PIL)

Библиотека для обработки цифровых изображений. Она позволяет открывать картинки, изменять их размер, накладывать водяные знаки, применять фильтры и сохранять в других форматах (например, из PNG в JPG).

from PIL import Image

# Открытие и изменение размера
img = Image.open("image.jpg")
img_resized = img.resize((200, 200))
img_resized.save("tiny_image.jpg")
