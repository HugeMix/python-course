Урок 14: Работа с датой и временем

Изучаем модуль datetime и способы форматирования даты и времени.

🎯 Цель урока

Познакомиться с модулем datetime и научиться получать, форматировать и вычислять разницу между датами.

📅 Модуль datetime

Python предоставляет встроенный модуль datetime для работы с датами и временем. Чтобы использовать его, нужно импортировать класс datetime из одноимённого модуля:

from datetime import datetime

now = datetime.now()
print("Текущая дата и время:", now)

⏳ Форматирование даты - метод strftime()

Метод strftime() конвертирует объект даты в строку по заданному шаблону. Название: string format time:

from datetime import datetime

now = datetime.now()
formatted_date = now.strftime("%d.%m.%Y %H:%M:%S")
print("Отформатированная дата:", formatted_date)
# Пример вывода: 02.03.2025 19:21:14

Основные символы форматирования:

  • %Y - год (четыре цифры, например 2025)
  • %m - месяц (01–12)
  • %d - день (01–31)
  • %H - час (00–23)
  • %M - минуты (00–59)
  • %S - секунды (00–59)

📆 Создание своей даты

Объект даты можно создать вручную, передав год, месяц, день и (опционально) время:

from datetime import datetime

custom_date = datetime(2025, 5, 17, 15, 30)
print("Созданная дата:", custom_date)
# Выведет: 2025-05-17 15:30:00

⏱️ Разница во времени - timedelta

Если вычесть одну дату из другой, получится объект timedelta - он хранит разницу в днях и секундах:

from datetime import datetime, timedelta

now = datetime.now()
deadline = datetime(2025, 12, 31)
delta = deadline - now
print("До конца года осталось:", delta.days, "дней")

С помощью timedelta можно также прибавлять или вычитать дни из даты:

from datetime import datetime, timedelta

today = datetime.now()
next_week = today + timedelta(days=7)
print("Через неделю будет:", next_week.strftime("%d.%m.%Y"))
